# DemoBlaze Playwright Automation Framework

This project demonstrates an automation framework built using Playwright with TypeScript.

Framework Design:
- Page Object Model (POM)
- Wrapper methods for reusable actions
- Test data management using JSON
- End-to-End automation scenario

Automated Scenario:
1. Login to Demoblaze
2. Navigate to Laptops category
3. Move to next page
4. Select MacBook Pro
5. Add product to cart
6. Verify product in cart
7. Delete product
8. Logout

Tech Stack:
- Playwright
- TypeScript
- Node.js
- Page Object Model

Run Tests:
npx playwright test
